Game Concept

The genre, overall concept of game

The genre of the game is an arcade style. An arcade style is a style where when all lives have been lost, the game restarts. Different levels are meant to challenge the user’s reflexes, and thinking ability.

The game is set in a medieval era. It involves a knight going through different levels to complete a ‘quest’. Each level give different tests that will challenge the player in different ways.
