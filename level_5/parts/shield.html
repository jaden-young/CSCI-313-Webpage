Shield

The shield is attached as a child object of the Player game object in the Unity editor. Since we handled the sword and shield as independently, each needed their own scripts and animations. We used the shield model from the knight asset from the asset store, while making our own custom animations and animation events.

Components of Shield:

  • Animator - Controls the shield animation. Only used when called by PlayerDefend script.
  • PlayerDefend Script - Simple script that controls raising the shield.
    • - PlayerHealth - Reference to player health script so we can set its isBlocking variable.
    • - void SetIsBlocking (int) - Called by Animation Events at the beginning and end of shield raising animation.
