Ranch Bottle

Variables

  • + bool gameOver - Set by an animation event when the ranch has finished rising. This is accessed by the LevelFiveManager script to end the game.

Methods

  • onPickup(): Starts the end game sequence when player picks up the bottle
  • + void RaiseBottle () - Causes the ranch to rise out of the chest.
