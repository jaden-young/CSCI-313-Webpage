Level Five Manager

This is a script attached to an empty game object in the scene that we use to end the game.

Variables

  • - BottleOfRanchController - Reference to ranch controller script that we use to access whether it has completed playing its animation.
  • - PlayerHealth - Reference to player script to see if the player has died.

Methods

  • - void Update () - Checks to see if either the player has died or if the ranch bottle has been obtained. If either are true, returns the player to the main menu.
