Chest

The chest has a box collider marked as a trigger attached to it, so that we can open the chest when the player comes into range.

Variables

  • - BottleOfRanchController Script - Reference to ranch controller so that we can play its animation, making it rise out of the chest

Methods

  • - void OnTriggerEnter (Collier other): Uses Unity's tag system to determine if it was the player that came in to range. If it was, opens the chest and raises the ranch bottle up, playing appropriate sound effects.
