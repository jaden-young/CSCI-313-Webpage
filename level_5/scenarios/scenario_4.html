Developer's Perspective

We use Unity's tag and message systems to control dealing damage. The script attached to the player's sword checks when it enters a collision if the object it collided with is tagged as a Dragon. Then, if the player is currently attacking (determined by Animation Events in sword), we use Unity's SendMessage to call the TakeDamage method on the dragon, passing in the damage we want to deal.
