Variables

Picture pictures[] -> This is a collection of picture Objects (4 total)

??? squence[] -> This is a random collection of pictures used to create the pattern the user have to repeat

Methods

Initialize() -> This is used to create the Object and initialize the variables

update() -> This method allows the Object to update and check for mouse clicks

Check() -> This allows the object to check if the clicked on picture is the correct one for the sequence. If the sequence is correct it will call SimonSaysManager's Correct method.

Show() -> This will show the squence to the user.

UpdateSequence() -> this is increase the sequence collection by 1 more object

delete() -> This method will delete the object when it is not longer needed
