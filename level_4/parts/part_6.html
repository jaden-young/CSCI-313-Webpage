Card

Variables

boolean flipped -> used to tell if the card is face up (true) or face down (false)

Methods

Initialize() -> This is used to create the Object and initialize the variables

Info() -> This method will return the value of the object so it can be used to compare

Flip() -> This allows the object to become either face up or face down. It will be used to let the system know if it is visable to the player or not

delete() -> This method will delete the object when it is not longer needed
