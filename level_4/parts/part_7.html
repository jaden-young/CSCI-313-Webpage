Level 4-Puzzle

SimonSaysManager

Variables

Board board -> This an instance of the Board Object.

integer tries -> This is the total number of tries for the puzzle that are left

integer puzzleScore -> This is the score achieved for this puzzle

Methods

Initialize() -> This is used to create the Object and initialize the variables

update() -> This method allows the Object to update and check for mouse clicks

Correct() -> This method is called when the correct sequence from Board is given. It is used to increase game score

Incorrect() -> This method is called when the incorrect sequence from the Board is given. It will take 1 try away and decide if it is the last try. It will also have the board display the sequence again.

delete() -> This method will delete the object when it is not longer needed
