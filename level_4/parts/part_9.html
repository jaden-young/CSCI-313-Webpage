Button

There are multiple of these in the game. Each is very similar with small differences which I will talk about at the end of the components.

Material

Each button has two different materials, the off material and the on material. The off material is when the button has not been 'clicked'. The on material is when the button has been clicked. The differences between the different buttons is just the color of the material. This is an example of the button on and off material.

Variables

Material Dark -> This material is the 'off' of the button

Material Light -> his material is the 'on' of the button

Methods

void LightUp() -> This method turns the light on

void Darken() -> This method turns the light off

Button Click Script

Variables

ButtonShow show -> instance of the ButtonShow script

float timeLift -> Meant to be timeLeft, used to tell the system when to turn the light off

bool turnedOn -> true when the button is turned on, false when it is off

SquenceMethods check -> instance of SquenceMethods ( in Board object) sends a number based on the certain button to the SquenceMethods to check if the correct one button for the sequence

Methods

void OnMouseDown() -> This is when the user clicks on the button, it is send to the check variable to see if it is the correct one

void Update() -> used to get input of the user is clicking on the object, it also used to decrease timeLift if needed

void showLight() -> called by SquenceMethods to show the button to the user
