Level 4-Puzzle

Memory Puzzle Starts

Player's Presepective

The player will have just finished colliding with the Puzzle Door. The player now see the board for the memory puzzle. Instructions will pop up.

Developer's Perspective

The Memory Manager will display the instructions for the puzzle. It will have been initialized which will randomly initialize 8 cards, each having a pair (4 pairs total). It will draw these cards face down on the board. The Memory Manager will start to update to watch for button clicks.
