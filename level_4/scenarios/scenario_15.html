Player's Presepective

Developer's Perspective

The level builder will notify the Game Manager (also known as the Scene Builder) that the level has been completed. Control will return to Game Manager. The Game Manager will delete the Level Builder which will delete the Avatar and Maze. The Game Manager will then be able to go onto the next level.
