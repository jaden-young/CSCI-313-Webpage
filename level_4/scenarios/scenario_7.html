Level 4-Puzzle

Player wins Memory

Player's Presepective

The player will have just flipped the last two cards, winning the puzzle. The score for the puzzle will be displayed to them and then they would be returned to the maze.

Developer's Perspective

Memory Manager will be notified that this was the last card to be paired up. It will find the total score of the puzzle by multiplying the tries left by the score received during the puzzle. It will then give that info to the Level builder to increase the overall score for the player. Memory Manager will then pass control back to Level Builder which will delete the Memory Manager. It will make the maze and avatar visible again and allow the player to continiue on in the maze.
