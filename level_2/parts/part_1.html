Avatar

Properties

  1. int NumberOfLifes - Indicate number of lives
  2. int Score - Indicate the current score for the level
  3. boolean Powerup - Indicate if the Avatar is currently under the effect of a PowerUp

Methods

  1. Initialize - Move Avatar to starting location, set number of lives to 3, score to 0, Powerup to false
  2. Update - Check with the system for movements/collisions
  3. Death - If hit by an enemy, check for PowerUp, if PowerUp is false(indicating no Powerup in effect) reduce the number of lifes by 1
  4. DisplayPowerUp - Will display the effect of the PowerUp indicated by the PowerUp class
